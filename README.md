Set individual scans in Tenable WAS.

Generate your ACCESS_KEY and SECRET_KEY in Tenable user settings.

use 1.testme_first.sh to check the connection and set up 1 example scan
```bash
bash 1.testme_first.sh
```


all.sh sets up all the scans in 1 bulk operation
mass_delete.sh removes the scheduled scans based on the scan name pattern
